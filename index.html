Sizes

Our typographic sizes are very flexible and use greek classes to indicate sizes. Each size also has margin and padding in keeping with the vertical rhythm of the page. If you prefer to use a cascading style, use typeset on your container div. This class will apply styles to semantic elements. For example, an h1 will be alpha-sized; an h2 will be beta-sized; p tags will be eta sized. Only use this element if the child elements are all typographic elements that don't need customization (you want to keep your code DRY by not having to override too many elements).

Buttons

To create a button, add the btn class to any a or button element. This base class must be used alongside an ordinality modifier (and must serve as the base for all other modifiers).

Ordinality

To indicate the a button represents a primary action, add the btn--primary class alongside btn. The available classnames are btn--primary, btn--secondary, and btn--accent. Accent buttons provide some extra visual weight over the other two and should be used sparingly.

Disabled

To indicate that a button is disabled, add the btn--disabled class alongside btn and an ordinality modifier.

Sizes

To change the size of a button, add the btn--small or btn--large class alongside btn and an ordinality modifier. The default size is regular.

Widths

The width-modifying class changes the button to be relative to the width of its container. To create a button that takes up the full width of its container, add the btn--full class alongside btn and an ordinality modifier. To create a button that takes up half the width of its container, add the btn--half class alongside btn and an ordinality modifier.
